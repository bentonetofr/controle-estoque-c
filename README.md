# Controle de Estoque em C

Sistema de gerenciamento de produtos desenvolvido em C, utilizando manipulação de arquivos binários.

## Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Busca por ID
- Remoção de produtos
- Validação de entrada
- Controle de IDs únicos

## Tecnologias
- Linguagem C
- Manipulação de arquivos (fopen, fread, fwrite)

## Como executar
gcc main.c -o estoque
./estoque
